Mais credibilidade Ao ativar a proteção SSL, o URL do seu site começará com HTTPS, e a barra de endereço também mostrará um ícone de cadeado. Esses elementos mostram aos visitantes que o seu site usa SSL e é confiável, desta forma, eles sabem que suas informações estão protegidas.

Nos acompanhe

    • 01

    Criptografa dados Se confiável, o navegador gerará duas chaves simétricas e enviará uma delas para o servidor web usando a chave pública. Assim que o servidor receber a chave, ele usará a chave privada para descriptografá-la. O navegador e servidor então estabelecem conexão segura para transferir dados.

    02

    Estabelece conexão segura Os sites receberão uma chave pública e privada assim que a compra de um certificado SSL seja bem sucedida. Quando um usuário acessar o site, o navegador solicitará o certificado SSL e a chave pública para verificar se o site é confiável.

    03

    Oferece indicadores visuais Um site com a instalação SSL bem sucedida terá vários indicadores na barra de endereço: um ícone de cadeado e um prefixo HTTPS no URL do site. Dependendo do tipo de certificado, os donos de sites podem também exibir um selo do site para aumentar a credibilidade.
